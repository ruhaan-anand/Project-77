Console Web App

Console screen is used to debug your code and check the output output of the your code step by step. You have to press Ctrl+Shift+I on your keyboard and click on console to open console screen. To print something on console screen you have to write console.log() in your js file and inside the round brackets, you have to write the thing that you want to print on console screen.

Visual Code studio is a offline software for coding and building websites. It is a offliine platform so you don't need internet to work on it. It is a really good platform because it has many shortcuts so it saves your time. To set it up you need to go to extensions and install only one extension called Live Server to see the output of your code on a different screen.
